jQuery返回顶部插件jQuery.toTop演示

简介

jQuery.toTop 是一款轻量级的 jQuery 返回顶部插件，压缩后不到 1KB。同时它也是可定制的，你可以设置是否自动隐藏、滚动时间、位置等等。

使用方法

1、引入文件

<script src="jsjquery.min.js"></script>
<script src="js/jquery.toTop.min.js"></script>

2、HTML

<a class="to-top">返回顶部</a>

3、JavaScript

$('.to-top').toTop();

配置

属性/方法 类型 默认值 说明
autohide 布尔值 true 自动隐藏
offset 整数 420 距离顶部多少距离时自动隐藏按钮
speed 整数 500 滚动持续时间
position 布尔值 true 如果设置为 false，则需要手动在 css 中设置“按钮”的位置
right 整数 15 右侧距离
bottom 整数 30 底部距离
